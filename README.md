# Клановый бот

## Настройка вопросов анкеты
Чтобы изменить/удалить вопросы в анекете, зайдите в файл vote.json и найдите переменную "questions". Там будут записаны базовые вопросы в формате:
```
"Вопрос1",
"Вопрос2"
```
